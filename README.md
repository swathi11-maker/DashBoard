Job Market Analysis Dashboard (Power BI)

1) Overview
This project presents an interactive Job Market Analysis Dashboard developed using Power BI.
The dashboard provides insights into salary distribution, AI adoption levels, automation risk, remote work trends, and job growth patterns across different industries and job roles.

2) Objectives
Analyze salary trends across job titles and industries
Evaluate AI adoption levels in organizations
Assess automation risk across job categories
Identify remote work patterns
Build an interactive and user-friendly dashboard

3) Dataset
The dataset includes the following attributes:
Job Title
Industry
Company Size
Location
AI Adoption Level
Automation Risk
Required Skills
Salary (USD)
Remote Friendly (Yes/No)
Job Growth Projection

4) Tools & Technologies
Power BI Desktop
DAX (Data Analysis Expressions)
Data Visualization Techniques

5) Dashboard Features
KPI Cards (Average Salary, AI Adoption Level, Automation Risk)
Bar Charts for salary comparison across industries
Donut/Pie Charts for category distribution
Scatter plots for hypothesis analysis
Interactive slicers (Job Title, Industry, Location)

6) Key Insights
Average salary is approximately $91,000 USD
Medium AI adoption level is most common
Automation risk varies across job roles
Significant portion of jobs are remote-friendly
Salary differs across industries and job titles
